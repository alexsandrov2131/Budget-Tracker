# Учёт трат

**Учёт трат** — Android-приложение для контроля личного бюджета с поддержкой тёмной темы.
Позволяет добавлять траты с указанием суммы и категории, просматривать список расходов
за выбранный месяц и видеть статистику по категориям. Все данные хранятся локально на устройстве.

## О проекте

Приложение выполнено в современном стиле с градиентным интерфейсом: фиолетово-изумрудная
палитра в светлой теме и тёмно-фиолетовая в тёмной. Интерфейс полностью на русском языке.

## Стек

- Java
- Android SDK (minSdk 26, targetSdk 34)
- Android Studio
- XML-разметка (без Jetpack Compose)
- SharedPreferences + Gson
- ConstraintLayout / LinearLayout / RecyclerView / CardView
- Material Components

## Возможности

- Splash-экран с градиентным фоном и кнопкой запуска
- Список трат за выбранный месяц с навигацией вперёд и назад
- Цветные метки категорий на карточках трат
- Добавление трат с указанием названия, суммы и категории
- Валидация суммы (число больше нуля)
- Удаление конкретной траты долгим нажатием
- Удаление всех трат за прошлые месяцы одной кнопкой
- Итоговая сумма трат за месяц
- Статистика по категориям за текущий месяц
- Переключение тёмной темы через настройки
- Полная поддержка тёмной темы на всех экранах

## Экраны

| Экран | Назначение |
|---|---|
| SplashActivity | Заставка с кнопкой «Начать» |
| MainActivity | Список трат за месяц + навигация по месяцам |
| AddExpenseActivity | Форма добавления траты |
| SettingsActivity | Тема |
| StatsActivity | Статистика по категориям за месяц |

## Структура проекта

```
app/src/main/java/com/budget/tracker/
├── model/
│   ├── Expense.java        — модель одной траты
│   └── Category.java       — фиксированный список категорий
├── util/
│   ├── ExpenseStorage.java — хранение трат в SharedPreferences (JSON через Gson)
│   └── SettingsStorage.java— хранение настроек темы
├── adapter/
│   └── ExpenseAdapter.java — адаптер RecyclerView для списка трат
└── activity/
    ├── SplashActivity.java
    ├── MainActivity.java
    ├── AddExpenseActivity.java
    ├── SettingsActivity.java
    └── StatsActivity.java
```

## Хранение данных

Все траты хранятся локально в SharedPreferences в формате JSON через библиотеку Gson.
Настройки темы также хранятся в SharedPreferences. Никакие данные не передаются на сервер.
